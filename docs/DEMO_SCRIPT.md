# Five Minute Demo Script

1. Dashboard: show one high-risk Black Friday case seeded from multi-source data.
2. Case detail: show statistical, relational, and behavior scores.
3. Active evidence retrieval: replay agent decisions and governed actions.
4. Evidence graph: explain device/IP/payment/refund/logistics linkage.
5. Evidence Passport: show support evidence, counter-evidence, uncertainty, and Human Review gate.
6. Pattern learning: show candidate pattern generated from trajectories and requiring human approval.
