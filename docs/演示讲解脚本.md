# 五分钟演示讲解脚本

1. **总览页**：展示三类典型跨境电商风险案件、证据数量、追证轨迹、OpenCLAW 动作调用和模型调用情况。
2. **案件卡片**：选择一个高风险案件，说明其统计异常、关系异常和行为异常分数。
3. **主动追证页**：播放智能体决策和受控动作执行过程，突出“下一步 action 由模型根据证据缺口选择”。
4. **多源融合图谱**：解释设备、IP、支付、退款、物流、评论和补贴台账如何形成证据图谱。
5. **证据护照页**：展示支持证据、反证、不确定性和人工复核门禁。
6. **模式学习页**：展示由轨迹归纳出的候选风险模式，以及人工确认后写回模式库和策略权重的闭环。
