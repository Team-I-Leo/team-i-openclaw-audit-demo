## 模式学习写回

已在 `runtime/aer_loop_model_smoke_9729073_openai_full_debug.sqlite` 上验证。

产品闭环支持：

- `pattern_learning_agent` 生成候选模式。
- 通过 `POST /api/patterns/candidates/{candidate_id}/review` 进行人工审核。
- 审核通过后写入 `risk_pattern`。
- 学习到的动作先验写入 `policy_action_weight`。
- 历史复用信息写入 `case_memory`。
- 路由器通过 `learned_policy_delta` 消费学习到的策略。

验证结果：

```json
{
  "review_risk_pattern_id": "LEARNED-EC-ARBI-003",
  "original_arbi_status": "active",
  "learned_pattern_written": true,
  "policy_weight_rows": 16,
  "case_memory_ids": [
    "MEM-EC-ARBI-003-EC-ARBI-003",
    "MEM-EC-ARBI-003-LEARNED-EC-ARBI-003"
  ],
  "route_uses_learned_policy": true
}
```

候选模式 ID 已做命名空间保护。如果模型提出的 ID 与生产模式冲突，例如 `EC-ARBI-003`，系统会将通过审核的学习模式写为 `LEARNED-EC-ARBI-003`，原始 `EC-ARBI-003` 保持 `active`。
