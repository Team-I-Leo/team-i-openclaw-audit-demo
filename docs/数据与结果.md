# 数据与结果

本仓库包含 Team-I 审计工作流所需的演示数据和精简验证输出，可以在不提交模型权重的前提下复现主要结果。

## 已包含内容

- `data/patterns/*.yaml`：初始风险模式定义。
- `runtime/aer_loop_model_smoke_9728967_full_debug.sqlite`：最终本地模型验证数据库。
- `runtime/pattern_learning_writeback_test.sqlite`：人工确认写回验证数据库。
- `runtime/aer_loop_model_quick_*.sqlite`：快速模型连通性验证数据库。
- `logs/model_summary_*.json`：精简验证摘要。
- `logs/model_smoke_run_*.json`：选定的完整运行输出。
- `materials/route_calibration_materials.tar.gz`：原始路线校准材料压缩包。

## 未包含内容

- `models/`：模型权重体积过大，不适合进入 GitHub，需要单独下载或在运行环境中挂载。
- `runtime/aer_loop_model_smoke_9729073_openai_full_debug.sqlite`：该原始 SQLite 文件在后续服务运行后超过 GitHub 单文件限制，因此不提交；对应验证摘要保留在 `logs/model_summary_9729073_openai_full_debug.json`。
- API 进程日志和 PID 文件：属于本地运行噪声，不影响结果复现。

## 回放主结果

```bash
export PYTHONPATH="$PWD/backend:${PYTHONPATH:-}"
export AER_DB_PATH="$PWD/runtime/aer_loop_model_smoke_9728967_full_debug.sqlite"
python -m aer_loop.cli summary
```
